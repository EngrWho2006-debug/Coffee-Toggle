# ☕ Glass UI Coffee Toggle Switch

A premium, interactive, and smooth fluid-morphing toggle switch built with pure HTML, CSS, and vanilla JavaScript. The switch transitions beautifully from a realistic coffee bean into a glassmorphism-styled coffee cup that dynamically fills up and releases animated steam.

## Features
- **Realistic Morphing:** Smoothly transforms from an organically shaped slim coffee bean into a perfect round cup.
- **Glassmorphic Track:** Modern Glass UI frosted look with a translucent background blur and clean light-reflection borders.
- **Dynamic Liquid Filling:** Coffee fluid slides up from the bottom only after the cup completes its rolling animation.
- **Delayed Steam Animation:** Three-layer animated steam waves that rise up sequentially after the coffee is fully filled.
